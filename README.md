# reciteQuestions
背题项目，若需要背诵相应东西想一个个输入回答，那么这个项目是你不二之选

技术涉及：NodeJs、Vue3、Vite、Html、JavaScript、SCSS。
注意：该项目与本地JSON数据建立连接服务器为nodeJS服务器！

警告：未经作者授权禁止转载!否则将保留追究法律责任！
